#GXBlog - 高效、简洁的博客系统#

##Info##

- Version 2.0.0

## Description ##

The PHP open source for creating your own blog easily. It's based on ThinkPHP 3.2.2 framework, if you know php, it's a nice way to rebuild the cms for customing with your need.
